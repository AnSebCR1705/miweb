## 🌱 Origin & Purpose

This project has its origins in the Data Mining (IND345) course made by the Dr. Vlada Pleshcheva and taught at the Faculty of Sciences and Engineering of the Pontifical University Catholic of Peru during the 2025-2 term. The original content of the course was taught in the statistical R language and is the property of both the university and the professor who was responsible for the course; therefore, the shared teaching material in this repository provides the datasets used during the academic cycle, as well as the reimplementation of the methodology used in the course, adapted to the Python programming language.

> 🎯 The purpose of this repository is to be viewed by students interested in learning about data science and so that they can transfer their knowledge of data analytics and machine learning techniques applied in R to Python, and to allow them to find the insights related to the differences between both languages.

### Course datasets (restricted)

- Some datasets were provided during the Data Mining course and are not publicly available. For this reason, they are not included in this repository.
- If you are a student from the university, you may request access using you institutional email:
